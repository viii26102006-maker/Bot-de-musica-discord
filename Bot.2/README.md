# Bot.2
Bot para escutar suas músicas. 
Ainda há de ser melhorado em alguns quesitos de comandos e restrutura, pois consiste ainda pequenos erros, mas é funcional, contendo uma forma de você fazer sua própria playlis usando os link do youtube.
## Comandos
j!entrar
j!tocar
j!pausar
j!continuar
j!pular
j!sair
## Comandos de para sua playlist pessoal
j!addplaylist
j!tocarplaylist
j!minha_playlist
